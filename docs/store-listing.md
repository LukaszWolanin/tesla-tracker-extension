## Short Description (132 chars max)

## Description (EN)

Track your Tesla vehicle delivery in real time — directly from your browser toolbar.

**Features:**
- Real-time delivery status with visual timeline
- Delivery window dates from Tesla's internal API
- Desktop notifications on status changes, VIN assignment, and delivery window updates
- Full vehicle configuration decoder (800+ option codes)
- Vehicle image from Tesla's configurator
- Automatic background polling (configurable interval)
- Auto-login with encrypted token storage
- Change history with deep-diff detection
- Works with Chrome, Edge, and Firefox

**Privacy-first:**
All data stays on your device. No external servers, no analytics, no tracking. Tokens are AES-256-GCM encrypted. Open-source code for full transparency.

**How it works:**
1. Click the extension icon and sign in with your Tesla account
2. Your order status, delivery window, and vehicle details appear instantly
3. The extension checks for updates in the background and notifies you of any changes

Perfect for anyone waiting for their Tesla delivery — never miss a status update again.
